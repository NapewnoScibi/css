Jakie jest znaczenie znaku #

Do zdefiniowania indyfikatora (ID).

Jak następuje wywołanie ID (wraz z wytłumaczeniem)?

Wywoływanie składa się z selektora do którego chcemy wpisać id

i znakiem # (przed nazwą) id="nazwa"

Przykład: id="#przyklad"

Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.

  • Moze wystopic tylko raz w calym dokumencie ( w przeciwienswie do klas)
  • Elementy oznaczone id muszą być unikatowe czyli na stronie moze byc jeden element z takim ID
  • Wieloktrone uzycie tego samego id nie wpływa na wydajnośc strony
