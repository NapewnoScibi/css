Budowa Selektora

  1. Wymień Dwie Czesci Selektora

    2. Selektor sklada sie z:

    • selektora

    • deklaracji

  2. Znaczenie nawisów klamrowym {...},

    3. W nawiasach klamrowych umieszcza się Deklaracje selektora CSS

  3. Opisz cechy,

    4. Jest częścią deklaracji , Jest to właściwośc lub własnośc poddana formatowaniu CSS

    Każda cecha ma ściśle wyszczególnioną listę wartości, które może przyjąć.

    Nalezy w niej wpisac konkretne atrybuty formatowania np font-family

  4. Opisz wartości

    5. Jest częścią deklaracji ,Jest to wartość która może się składać np z liczby

    Wpisujemy tam wartośc atrybutu np green

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    6. Znak średnika stosujemy kiedy chcemyzakonczyc deklaracje

    Nie jest on konieczny kiedy jest tylko jedna deklaracja
